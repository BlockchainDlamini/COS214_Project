\doxysubsection{Detailed Description}
Represents a chef in the kitchen. 

The \mbox{\hyperlink{classBaseChef}{Base\+Chef}} class is a part of a larger kitchen management system. It is the base class from which other chef classes inherit. It provides common functionality and serves as a foundation for more specialized chefs.
